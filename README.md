# PMYO AI Assistant (Android Chatbot)

### US Project Overview
A native Android application developed with **Kotlin**, designed to assist university students. The app integrates **Google Gemini API** for generative AI capabilities and uses **Firebase** for user authentication and data storage.

* **Status:** Completed
* **Architecture:** MVVM Pattern (Model-View-ViewModel)
* **Tech Stack:** Kotlin, XML, Firebase (Auth & Firestore), Gemini API, Volley.
